## Features

- Search for movies
- Read detailed information about each movie
- Watch trailers
- Make lists of your favorite movies and save them for later
- Rate movies
- Filter on popularity, genre, rating etc.
- Multi-language
- Offline available
- Nightmode
- Share movies with your friends
